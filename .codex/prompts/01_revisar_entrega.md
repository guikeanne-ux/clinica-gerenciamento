# Prompt — revisar uma entrega já implementada

Leia `AGENTS.md`.

Revise a entrega atual comparando o código com:

`specs-entregas/XX_NOME_DA_ENTREGA.md`

Faça uma revisão crítica e objetiva.

Verifique:
- escopo implementado;
- coisas implementadas fora de hora;
- arquitetura modular;
- UUID nas APIs;
- validações;
- permissões;
- auditoria;
- testes;
- lint;
- PHPStan;
- migrations;
- frontend, se aplicável;
- segurança.

Não implemente nada ainda.

Responda com:

```md
## Revisão da entrega

### Aprovado

### Problemas bloqueantes

### Problemas importantes

### Melhorias recomendadas

### Arquivos que precisam ser corrigidos

### Comandos que devem ser rodados
```
