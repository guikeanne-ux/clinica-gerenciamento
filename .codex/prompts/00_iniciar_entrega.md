# Prompt — iniciar uma entrega

Leia primeiro o arquivo `AGENTS.md`.

Depois leia o arquivo:

`specs-entregas/XX_NOME_DA_ENTREGA.md`

Implemente somente essa entrega.

Regras:
- Não implemente funcionalidades de entregas futuras.
- Não altere a stack.
- Não use Laravel.
- Não use framework frontend.
- Não exponha `id` numérico em APIs.
- Use UUID.
- Crie migrations, seeders/factories e testes quando aplicável.
- Rode os comandos possíveis.
- Corrija erros reais.

Ao terminar, responda com:

```md
## Entrega concluída

### Arquivos criados/alterados

### Como rodar

### Como testar

### Resultado dos comandos executados

### Pendências reais

### O que não foi implementado por pertencer a etapas futuras
```
